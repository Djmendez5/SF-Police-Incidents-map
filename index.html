Project Description

What I hope to achieve with this project is to paint a larger picture in the crime statistics in San Francisco
by visualizing the more common crime in different areas of the city, police district, and time of the incident
as well as noting if there is any strong correlation between the area and the crime, studying if certain areas are
more prone to specific crimes and so authorities can study if there is an underlining reason for such correlations.

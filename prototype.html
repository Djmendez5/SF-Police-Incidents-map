prototype 1

The Goal for the project is for the users to be able to easily look at the crime data in San Francisco easily and quickly, using the color legend to easily see what type of crimes are going on in different areas in San Francisco and be able to quickly get more details about each incident between January and May 2018.
The interactively I have in mind is to allow details on demand, the details would include: Category, short description, date, time, police district and the actual address. Since the crime category's legend will be displayed the user should be able to click on a certain category and then filter for that catergory, displaying only those marks.
